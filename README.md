## :rocket: Tecnologias

- [Node Js](https://nodejs.org/en/)
- [Nunjucks](https://mozilla.github.io/nunjucks/)

## :fire: Instalação
```bash
$ git clone https://github.com/ClodoaldoDantas/gym-academy.git
$ cd gym-academy
$ npm install
$ npm run dev
```
